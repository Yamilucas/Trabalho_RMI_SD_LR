Projeto desenvolvido para a disciplina de **Sistemas Distribuídos**, implementado utilizando **Java RMI** para comunicação entre cliente e servidor em um ambiente distribuído.

## Descrição do Projeto

O sistema consiste em uma aplicação distribuída de **Vendas e Controle de Estoque para uma Farmácia**, estruturada em duas partes distintas:

### Cliente
Aplicação com interface gráfica desenvolvida em **Java** utilizando **JFrame (Swing)**.

### Servidor
Responsável pelas regras de negócio, processamento das requisições remotas e comunicação com o **banco de dados SQL**.

### Estrutura de Execução

- Um computador executando o **cliente (interface gráfica)**
- Outro computador executando o **servidor (banco de dados e serviços RMI)**
